What's this program for?

This is a tool for band alignment simulation considering strain effect in a heterostructure of III-V materials. You can chose three material name from the form boxes and input compound material composition in 0.0 ~ 1.0 where x is for a ternary material while, y is for a quaternary material. hence, if you want to get a binary or a ternary material band alignment, you can just inpiut only x value.
